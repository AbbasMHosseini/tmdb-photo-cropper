# TMDB Photo Cropper

A fast tool for preparing 2:3 TMDB person profile photos.

## Current version

This is the first MVP.

It supports:

- Uploading an image
- Loading an image URL
- 2:3 portrait crop preview
- Drag to reposition
- Zoom control
- Reset
- JPG export
- Export sizes: 500 x 750, 1000 x 1500, 1500 x 2250

TMDB checking and face detection are mocked for now. The structure is ready for real implementation later.

## Online version

This project is configured for GitHub Pages deployment through GitHub Actions.

## Run locally

```bash
npm install
npm run dev
```

Then open the local URL shown in Terminal.
